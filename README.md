To run DNArepeats.py script place genomic DNA of _Listeria monocytogenes_ genom in C:/LM.txt catalog. Default items like genome, length of repeated DNA sequence motif and andjustement of minimal number of repetitions can be modified by user selection in the DNArepeats.py script. Analysis of the large genomes may require strong processors and large RAM memory capacity. File DNArepeatsUr.exe you can run in Windows system.
